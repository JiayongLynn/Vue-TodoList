# vue-simple-todolist
已实现新增，删除代办事项，还可以将事项的完成状态改变，另外还有算出还剩多少项没有完成。
同时使用vue-router跳转到已过滤的菜单。
采用localStorage储存。
第一个Vue练手的TodoList项目，把一些基础的知识点运用了。
